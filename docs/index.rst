Welcome to weebapi's documentation!
=====================================

This is a API wrapper library for invite only service called weeb.sh. This library provides a modern, pythonic and
easy to use, object driven, interface.

This API wrapper also provides full discord.py rewrite integration. Check examples or Client object for more information.
